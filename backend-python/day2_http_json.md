basic set of methods that are used to interact with server:
1. GET : retrieve a resource or info
2. POST : interact a particular resource
3. PUT : exchange a resource
4. DELETE : delete a resource
5. HEAD : no body message only header is returned
6. PATCH : change a particular part of resource
7. OPTION : what options are available

HTTPS status code:
1xx : infromational
2xx : Success 
3xx : Redirection
4xx : client error
5xx : server error
